Timesheet Summary

Whats this presentation about?

This is a summary of some dummy timesheet data to illustrate how the data can be visualised using an interactive presentation using R and Shiny.

The presentation imports a CSV file of the dummy timesheet data, creates a plot for each aspect (week, person, project) and presents a slide per plot, allowing the user to filter the data.

What aspects of timesheet data will we review?

  • By Week
  • By Person
  • By Project
